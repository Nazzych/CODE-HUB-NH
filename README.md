## Run project:
```
- For run the project you need:
  Backend:
    1. Download the project on your computer;
    2. Go in "backend/" folder;
    3. Run in terminal: pip install -r requirements.txt ;
    4. After success proceіs you need run: python manage.py runserver ;
  Frontend:
    1. Go in "frontend/" folder;
    2. Run in terminal: npm start ;
    3. If you got errors maybe you don't have installed the Node.js (https://nodejs.org/uk), please install pakage and then errors must inihilate.
```
